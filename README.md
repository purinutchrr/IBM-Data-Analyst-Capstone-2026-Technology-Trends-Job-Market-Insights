## Project Overview
This project is the final capstone for the **IBM Data Analyst Professional Certificate**. It provides a comprehensive, data-driven analysis of the technology job market and developer trends for 2026. By leveraging various data collection methods and analytical tools, this project aims to help stakeholders make informed decisions regarding hiring, training, and strategic upskilling.

## Key Market Insights (2026 Forecast)
* **Leading Job Hub:** Washington DC is the top-tier tech hub with **5,316 active job postings**.
* **Highest ROI Skills:** **Swift** leads as the highest-paying programming language, averaging **$130,801/year**.
* **Database Dominance:** **PostgreSQL** remains the industry favorite, ranking #1 in both current usage and future desire.
* **The "TypeScript Surge":** A significant shift is occurring as developers move toward **TypeScript**, **Rust**, and **Go** for better performance and safety.
* **Cloud Infrastructure:** The market is dominated by the "Big Three" providers: **AWS**, **Azure**, and **Google Cloud**.

## Technological Stack
* **Programming Languages:** Python, SQL
* **Data Wrangling:** Pandas, NumPy
* **Web Scraping:** BeautifulSoup, Requests
* **Data Visualization:** Matplotlib, Seaborn, Plotly
* **Database Engine:** SQLite (Relational EDA)
* **BI Tool:** Google Looker Studio

## Project Structure
The project follows a standard data analysis lifecycle, organized into 6 core modules:

| Module | Phase | Key Tasks & Deliverables |
| :--- | :--- | :--- |
| **Module 1** | **Data Collection** | Collecting technical job data via Flask APIs and scraping salary benchmarks using BeautifulSoup. |
| **Module 2** | **Data Wrangling** | Cleaning and standardizing **18,845 records**, handling duplicates, and managing missing values. |
| **Module 3** | **Exploratory Data Analysis** | Utilizing SQL (SQLite) for deep-dive profiling into developer demographics and tech stack usage. |
| **Module 4** | **Data Visualization** | Creating distribution plots (Box plots, Histograms) to analyze market trends. |
| **Module 5** | **Dashboard Development** | Building an interactive 3-tab executive dashboard in Google Looker Studio. |
| **Module 6** | **Strategic Reporting** | Synthesizing findings into a final presentation with actionable recommendations. |
