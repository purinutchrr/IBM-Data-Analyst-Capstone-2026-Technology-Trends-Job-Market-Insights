## Project Overview
This project is the final capstone for the **IBM Data Analyst Professional Certificate**. [cite_start]It provides a comprehensive, data-driven analysis of the technology job market and developer trends for 2026 [cite: 44-45]. ## Key Market Insights (2026 Forecast)
* [cite_start]**Leading Job Hub:** Washington DC is the top-tier tech hub with **5,316 active job postings** [cite: 27-28].
* [cite_start]**Highest ROI Skills:** **Swift** leads as the highest-paying programming language, averaging **$130,801/year**[cite: 29].
* [cite_start]**Database Dominance:** **PostgreSQL** remains the industry favorite, ranking #1 in both current usage and future desire[cite: 30, 205].
* [cite_start]**The "TypeScript Surge":** A significant shift is occurring as developers move toward **TypeScript**, **Rust**, and **Go** for better performance and safety[cite: 31, 154].
* [cite_start]**Cloud Infrastructure:** The market is dominated by the "Big Three" providers: **AWS**, **Azure**, and **Google Cloud**[cite: 285].
